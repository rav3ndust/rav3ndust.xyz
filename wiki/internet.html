internet

The internet needs no introduction, at least at a conceptual level. You know what it is - after all, you're on it right now!

There are so many layers to the internet, though, and a lot of ways that you can avoid using the services of the "corporate web", as I like to call it. If you want to check out more decentralized and "smallweb" things, this next area is for you to check out.


relevant links

  • browsers: check out some of the FOSS browsers you can use as your gateway to the web
  • fediverse: A federated network of decentralized social platforms
  • gemini: a protocol for a simple, minimal web
  • matrix: a federated communication protocol
  • nostr: a simple protocol for global, decentralized, and censorship-resistant social media
  • tor: an overlay network to empower anonymous communication
  • xmpp: an open and independent communication standard
