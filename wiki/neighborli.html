neighborli

An open-source and federated social network for friends and freedom.

neighborli is an open-source social network on the Fediverse.

Neighborli is open to everyone who wants to join - just visit the URL, fill in the details on the signup page, and start following some accounts to build your feed. Once you feel comfortable, begin making some posts! If you have any questions, feel free to message myself at this link or the other instance admin, Ryoko, for help or support requests.

If you have noticed a bug within neighborli, please report it on the issue tracker at this link.

specs:

Here is a bit of the tech stack behind Neighborli:

  • Instance type: Mastodon
  • Web server: nginx
  • Server OS: Debian
