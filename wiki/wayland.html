wayland

the modern display server for the free desktop

Wayland is the modern replacement for the X11 window system. More and more, it is what major distributions are switching to for the default experience, and many X11-based desktop environments either have excellent Wayland compatibility at this point (GNOME/KDE), or are actively working on their Wayland implementations (Cinnamon, Xfce, MATE, to name a few).

Window managers are not left out here, either. If you like minimal window manager setups as I do, you will find some great options in Sway (which the wayland version of wiredWM is based on), dwl, niri, river, and others!

Desktops that speak the Wayland protocol are called "compositors".


relevant links

Wayland Book: A great primer on the Wayland protocol and how it works, by Drew Devault.

wlroots: A modular library for building Wayland compositors.

We Are Wayland Now!: A handy resource showing Wayland-compatible alternatives for common X11 tools.
