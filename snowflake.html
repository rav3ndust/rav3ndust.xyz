This page allows you to run a snowflake proxy to help people bypass censorship with the Tor network.

Snowflake proxies are run by volunteers, and users get matched up with random proxies, which are able to be run by anyone.

Open this page up whenever you want, turn on the proxy, and simply leave it open until you're ready to close it. Thanks for helping people who may be censored from using Tor.

simply toggle the button to enable the snowflake proxy!


browser extension

There is a Snowflake browser extension you can use on Chromium and Firefox-based browsers that acts the same way as the widget on this page, except it's right there in your browser toolbar. Here are the links for Firefox and Chromium if you want to grab it:

  • snowflake firefox extension
  • snowflake chromium extension

You can also enable Snowflake in the Brave Browser by going to brave://settings/privacy and enabling the Volunteer to help others connect to the Tor Network option.
